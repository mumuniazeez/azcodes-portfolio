## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/mumuniazeez/azcodes-portfolio.git

# Navigate into the project
cd azcodes-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
